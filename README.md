# Advent of Code 2020 (selected days)

## Setup
Install Node.js, then run each day from its folder or from repo root.

## Run (recommended)
Each day reads `./input.txt` in its day folder by default.

node day02/index.js
node day04/index.js
node day05/index.js
node day07/index.js
node day11/index.js
node day12/index.js
node day13/index.js

## Optional: pass an explicit input file
node day02/index.js path/to/input.txt
